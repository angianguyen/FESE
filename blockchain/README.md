# FESE Blockchain

Smart contracts cho dự án FESE (Financial Engineering & Smart Economy).

## Cài đặt

```bash
npm install
```

## Biên dịch Smart Contracts

```bash
npm run compile
```

## Chạy Local Blockchain Node

```bash
npm run node
```

## Deploy Smart Contracts

### Deploy lên local network
```bash
npm run deploy:local
```

## Kiểm tra

```bash
npm test
```

## Cấu trúc

- `contracts/` - Solidity smart contracts
- `scripts/` - Deployment và interaction scripts
- `tests/` - Contract tests
- `hardhat.config.js` - Hardhat configuration
